# Splorin Squirt Landing Pages

Three static HTML directions for GitHub Pages:

- `wine-vintage.html`: fake fancy wine vintage
- `startup-landing.html`: classic product startup parody
- `stealth-ai.html`: fake stealth AI company

Each page references the images in `assets/`. To publish one as the main GitHub Pages page, rename that file to `index.html` and keep the `assets/` folder next to it.
